# C01Lab6

## Useful Commands
Be sure to use ```npm install``` before any backend runs with npm.

### Launch the Mongodb Backend
```mongod --dbpath=data/db```

### Connect to Backend
```npm run dev```

### To Run Testcases
```npm test```

### Docker Verifying Identity on Linux
```gpg --generate-key```, then ```pass init <your-generated-key>```
